
  PROGRAM: p123.f90

  p123 performs the 3D analysis of Laplace's equation. The program is a 
  modified version of the one published in Smith I.M. and Griffiths D.V., 
  "Programming the Finite Element Method", 4th Edition, Wiley, 2004.

    Usage: p123 <job_name>

  The original program from the book, p123.old.f90 has been modified slightly
  to work with this version of ParaFEM.
