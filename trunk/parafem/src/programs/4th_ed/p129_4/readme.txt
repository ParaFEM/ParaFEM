  PROGRAM: p129_4.f90

  p129_4 solves the forced vibration of a 3d elastic solid, using a lumped or
  consistent mass. The program is a modified version of the one published in 
  Smith I.M. and Griffiths D.V., "Programming the Finite Element Method", 
  4th Edition, Wiley, 2004.

    Usage: p129_4 <job_name>
