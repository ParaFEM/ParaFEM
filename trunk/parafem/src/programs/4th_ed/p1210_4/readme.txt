  PROGRAM: p1210_4.f90

  p1210_4 solves the forced vibration of an elastic-plastic(Von Mises) solid
  It uses the viscoplastic strain method, with a lumped mass and explicit 
  integration. The program is a modified version of the one published in 
  Smith I.M. and Griffiths D.V., "Programming the Finite Element Method", 
  4th Edition, Wiley, 2004.

    Usage: p1210_4 <job_name>
