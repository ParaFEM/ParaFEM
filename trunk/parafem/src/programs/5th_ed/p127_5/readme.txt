
  PROGRAM: p127_5.f90

  p127_5 solves the 3D steady state Navier-Stokes equations. 
  This program is identical to the one published in Smith IM, Griffiths DV and 
  Margetts L "Programming the Finite Element Method", 5th Edition, Wiley, 2013.

    Usage: p127_5 <job_name>

  This was originally program p126 in the 4th Edition of the book.
