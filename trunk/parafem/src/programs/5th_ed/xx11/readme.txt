
  PROGRAM: xx11.f90

  xx11 performs the 3D analysis of Laplace's equation using 8-node brick
  elements. Preconditioned conjugate gradient solver. Parallel version.
  The program is a modified version of p123.f90 published in  
  Smith I.M. and Griffiths D.V., "Programming the Finite Element Method",
  4th Edition, Wiley, 2004.

    Usage: xx11 <job_name>
