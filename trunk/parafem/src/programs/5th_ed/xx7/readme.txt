
  PROGRAM: xx7.f90

  xx7 performs the 3D analysis of a linear elastic solid and adds support for 
  large deformations.

    Usage: xx7 <job_name>

  *** UNDER DEVELOPMENT ***

  This program uses an old version of ParaFEM. Modifications are in progress
  to get it working with this version of ParaFEM.
