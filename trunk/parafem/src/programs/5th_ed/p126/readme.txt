  PROGRAM: p126.f90

  p126 solves the 3D steady state Navier-Stokes equations. This program is 
  identical to the one published in Smith IM, Griffiths DV and Margetts L 
  "Programming the Finite Element Method", 5th Edition, Wiley, 2014.
  Please do not commit any changes to this program.

    Usage: p126 <job_name>
