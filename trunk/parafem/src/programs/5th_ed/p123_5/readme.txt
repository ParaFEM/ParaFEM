
  PROGRAM: p123_5.f90

  p123_5 performs the 3D analysis of the steady state heat equation. This 
  program is identical to the one published in Smith IM, Griffiths DV and 
  Margetts L "Programming the Finite Element Method", 5th Edition, Wiley, 2013.

    Usage: p123_5 <job_name>

  This was originally program p123 in the 4th Edition of the book and is 
  similar to program xx11 which is still under development by Llion Evans in 
  folder /dev
