
  PROGRAM: p123_5.f90

  p123_5 performs the 3D analysis of a linear elastic solid. The program is a 
  modified version of the one published in Smith I.M. and Griffiths D.V. 
  and Margetts L. "Programming the Finite Element Method", 5th Edition, Wiley, 2013.

    Usage: p123_5 <job_name>

  This was originally program xx5 in folder /dev
