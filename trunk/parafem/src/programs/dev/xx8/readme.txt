
  PROGRAM: xx8.f90

  xx8 performs the 3D analysis of a linear elastic solid. The first version 
  of this program is identical to the one published in Smith I.M., 
  Griffiths D.V. and Margetts L. "Programming the Finite Element Method", 
  5th Edition, Wiley, 2014. 

  The program is a development version being used to test potential
  optimisations on the BlueGene/Q system Joule at the Hartree Centre.

    Usage: xx8 <job_name>
