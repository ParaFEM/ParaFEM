  PROGRAM: xx16.f90

  xx16 performs the 3D analysis of a linear elastic solid. The program is 
  based on the one published in Smith I.M. and Griffiths D.V. and Margetts
  L. "Programming the Finite Element Method", 5th Edition, Wiley, 2014.

    Usage: xx16 <job_name>
