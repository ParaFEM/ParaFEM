
  PROGRAM: xx12_b.f90

  xx12_b is a branch of xx12

  xx12_b performs the 3D analysis of conduction equation using 8-node brick
  elements. Preconditioned conjugate gradient solver. Implicit
  integration in time using 'theta' method. Parallel version.
  The program is a modified version of p124.f90 published in  
  Smith I.M. and Griffiths D.V., "Programming the Finite Element Method",
  4th Edition, Wiley, 2004.

    Usage: xx12_b <job_name>
