
  PROGRAM: gaussianlds.py

  gaussianlds is a python script that creates thermal flux boundary conditions to
  create a laser flash analysis simulation to be used with prograx dev/xx12.

    Usage: gaussianlds.py <filename>
  
           e.g., gaussianlds.py cylinder

 NOTES 

  o The program has not yet been finished
  o Filenames etc are currently hard coded into script, needs generalising.
  o Script also produces .time file, change parameters within script if desired.
