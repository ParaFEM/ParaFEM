TTRB2TTR

Converts the binary temperature file .ttrb output from program xx12.f90 into
an ascii temperature file .ttr

USAGE: ttrb2ttr <job_name>
       expects <job_name>.ttrb and <job_name>.npp as input files
       outputs <job_name>.ttr
