
PROGRAM: p126.f90

The files p126-tiny.{bnd,d,dat,lid} comprise an example input deck to use with 
program p126 in ParaFEM.

To view the results in the ParaFEM-Viewer, the output file p126-tiny.upvw needs
to be post processed with the tool upvw2dis. This converts the .upvw file into 
a .dis file.
