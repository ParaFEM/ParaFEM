
The files in this directory are input files for the program sg12mg.f90

Using the input file p128ar-<size>.dat, program sg12mg.f90 will create a full 
input deck for program p128ar. The input deck will be for the example test 
problem defined for program p128 in Smith and Griffiths "Programming the Finite
Element Method."

Note: program p128ar.f90 is not in the Smith and Griffiths text.
