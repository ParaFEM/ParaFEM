
The files in this directory are input files for the program mg2d.f90

Using the input file p1212-<size>.dat, program mg2d.f90 will create a full 
input deck for program p1212. The input deck will is for the example test 
problem defined for program p128 in Smith and Griffiths "Programming the Finite
Element Method."

Note: program p1212.f90 is not in the Smith and Griffiths text.
