
PROGRAM: p126_5.f90

The files p126_5_tiny.{bnd,d,dat,lid} comprise an example input deck to use with 
program p126_5 in ParaFEM.

To view the results in the ParaFEM-Viewer, the output file p126_5_tiny.upvw needs
to be post processed with the tool upvw2dis. This converts the .upvw file into 
a .dis file.
