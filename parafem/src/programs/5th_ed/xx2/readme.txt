
  PROGRAM: xx2.f90

  xx2 performs the 3D analysis of a linear elastic solid. The program is a 
  modified version of the one published in Smith I.M. and Griffiths D.V., 
  "Programming the Finite Element Method", 4th Edition, Wiley, 2004.

  This program supports multiple material types and uses a mixed mode 
  parallel computing strategy with OpenMP and MPI.
  
  *** Warning: Under development ***
  
    Usage: xx2 <job_name>
