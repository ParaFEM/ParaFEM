NDTTRGET

Read temperature values from .ensi.NDTTR files (output from program xx12.f90)
for a specific node and writes to .ndttr

USAGE: ndttrget <job_name> <nstep> <nres> <format>
       expects <job_name>.ensi.NDTTR-****** as input files
       outputs <job_name>.ndttr
