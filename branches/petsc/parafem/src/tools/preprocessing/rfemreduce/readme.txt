
  PROGRAM: rfemreduce.f90

  rfemreduce is a simple program that loads a set of result data files that hold the number of nodes
  in that run that are over a given von mises threshold. The result is a set of histogram counts for
  a given number of bins.
  
    Usage: rfemreduce <job_name> <num_instances> <num_bins>
