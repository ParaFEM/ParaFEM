
  *** READ ONLY PROGRAM ***

  PROGRAM: p127.f90

  p127 solves the consolidation of a cuboidal Biot elasticvsolid using 20-node
  solid hexahedral elements coupled to 8-node fluid elements. This program is 
  identical to the one published in Smith IM, Griffiths DV and Margetts L 
  "Programming the Finite Element Method", 5th Edition, Wiley, 2014.
  Please do not commit any changes to this program.

    Usage: p127 <job_name>
