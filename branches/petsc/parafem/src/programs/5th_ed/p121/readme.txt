
  *** READ ONLY PROGRAM ***
  
  PROGRAM: p121.f90

  p121 performs the 3D analysis of a linear elastic solid. The program is 
  identical to the one published in Smith I.M. and Griffiths D.V. and Margetts
  L. "Programming the Finite Element Method", 5th Edition, Wiley, 2014. Please
  do not commit any changes to this program.

    Usage: p121 <job_name>
