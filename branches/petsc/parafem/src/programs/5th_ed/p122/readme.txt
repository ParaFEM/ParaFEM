
  *** READ ONLY PROGRAM ***
  
  PROGRAM: p122.f90

  p122 performs the 3D analysis of an elastic-plastic (Mohr-Coulomb) solid 
  using the viscoplastic strain method. The program is identical to the one
  published in Smith I.M. and Griffiths D.V. and Margetts L. "Programming the
  Finite Element Method", 5th Edition, Wiley, 2014. Please do not commit any
  changes to this program.

    Usage: p122 <job_name>
