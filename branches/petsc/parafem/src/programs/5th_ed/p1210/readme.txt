  PROGRAM: p1210.f90

  p1210 solves forced vibration of an elastic-plastic(Von Mises) solid. This 
  program is identical to the one published in Smith IM, Griffiths DV and 
  Margetts L "Programming the Finite Element Method", 5th Edition, Wiley, 2014.
  Please do not commit any changes to this program.

    Usage: p1210 <job_name>
