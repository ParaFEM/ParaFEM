  PROGRAM: xx18.f90

  xx18 performs the 3D analysis of an elastic solid using the PETSc solver 
  library.

  The program is based on program p121 published in Smith IM, Griffiths DV 
  and Margetts L "Programming the Finite Element Method", 5th Edition, Wiley, 
  2014.

    Usage: xx18 <job_name>
