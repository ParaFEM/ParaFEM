  PROGRAM: xx17.f90

  xx17 performs the 3D steady state solution of the Navier Stokes equations.
  The program is based on the one published in Smith I.M. and Griffiths D.V. 
  and Margetts L. "Programming the Finite Element Method", 5th Edition, Wiley, 
  2014.

    Usage: xx17 <job_name>
